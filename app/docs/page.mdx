# Welcome to Dex UI

Dex UI is a powerful and flexible animation UI components library designed to help developers create engaging and dynamic web applications with ease. Whether you're building a new project or enhancing an existing one, our library provides a suite of customizable components that will make your user interface come alive.

## Why Dex UI?

We understand the challenges developers face when integrating animations into their web projects—from complex logic to performance concerns. That’s why we created Dex UI: to simplify the process of implementing high-quality, responsive animations that elevate your web applications. With easy-to-use API, responsive defaults, and extensive customization options, our library lets you focus on building your project instead of spending time on complex code.
